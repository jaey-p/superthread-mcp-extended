All MCP Clients

Installation examples for Superthread MCP Extended clients

This MCP server (Superthread Extended) supports all MCP clients. Below are configuration examples.

Note: Replace YOUR_API_KEY with your Superthread Personal Access Token (PAT) prefixed with stp-.

MCP Endpoint URL (Use this for setup): https://superthread.sorbet.studio/mcp/app

Claude Code

▼

Run this command. See Claude Code MCP docs for more info.

Remote Server Connection

claude mcp add --header "Authorization: Bearer stp-YOUR_KEY" --transport http superthread https://superthread.sorbet.studio/mcp/app

Local Server Connection (Development)

If you have cloned the repository locally:

claude mcp add superthread -- bun run src/index.ts

Cursor

▼

Go to: Settings -> Cursor Settings -> MCP -> Add new global MCP server

Pasting the following configuration into your Cursor ~/.cursor/mcp.json file is the recommended approach. Use Cursor MCP docs for more info.

Remote Server Connection

{
  "mcpServers": {
    "superthread": {
      "url": "https://superthread.sorbet.studio/mcp/app",
      "headers": {
        "Authorization": "Bearer stp-YOUR_KEY"
      }
    }
  }
}

VS Code

▼

Add this to your VS Code MCP config file. See VS Code MCP docs.

Remote Server Connection

"mcp": {
  "servers": {
    "superthread": {
      "type": "http",
      "url": "https://superthread.sorbet.studio/mcp/app",
      "headers": {
        "Authorization": "Bearer stp-YOUR_KEY"
      }
    }
  }
}

Windsurf

▼

Add this to your Windsurf MCP config file. See Windsurf MCP docs.

Remote Server Connection

{
  "mcpServers": {
    "superthread": {
      "serverUrl": "https://superthread.sorbet.studio/mcp/app",
      "headers": {
        "Authorization": "Bearer stp-YOUR_KEY"
      }
    }
  }
}

Claude Desktop

▼

Remote Server Connection

Open Claude Desktop and navigate to Settings > Connectors > Add Custom Connector.

Enter the name as Superthread and the remote MCP server URL as https://superthread.sorbet.studio/mcp/app.

Alternative Configuration

Add to your claude_desktop_config.json:

{
  "mcpServers": {
    "superthread-extended": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote@latest",
        "https://superthread.sorbet.studio/mcp/app",
        "--header",
        "Authorization:${ST_PAT}"
      ],
      "env": {
        "ST_PAT": "Bearer stp-YOUR_KEY"
      }
    }
  }
}
